# Prerequisites

## Software you need

- **Claude Code** — https://claude.com/product/claude-code
- **Claude.ai account** — free to start, Pro ($20/mo) recommended by Day 4, Max ($100+) by Day 10.
- **Git** — for this repo
- **A terminal** — Warp is what Samin uses
- **A code editor** — VS Code works great with Claude Code

## Software you'll install as you go

Don't front-load these. The course walks you through each one.

- Whisperflow (voice)
- Warp terminal (Day 16)
- Obsidian (Day 16)
- Composio (Day 17)
- HeyGen (Day 26)
- Blotato (course)
- VibeFrame (Day 27)

## Accounts you'll make as you go

- Anthropic API (for some days)
- Alpaca (paper trading — Day 11+)
- Capital Trades / Unusual Whales (optional — Day 13)
- Composio (Day 17)
- HeyGen (Day 26)

All optional based on what you care about.

## Hardware

- Any Mac or Windows machine from the last 3 years.
- Keep your OS updated. Claude Desktop app moves fast.
- A phone for Day 8 / Day 18 (Telegram + Discord pairing).

## Time

- 15–20 minutes per day for 28 days = ~8 hours total, paced over a month.
- Or cram weekends if that's your style.
