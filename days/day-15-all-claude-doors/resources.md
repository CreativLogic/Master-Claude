# Day 15 — Pick the Right Claude for the Job

## Tools used in the video

- **VS Code** — https://code.visualstudio.com/ — Free editor every developer uses. Install the Claude extension to use Claude right against your files.
- **Claude for VS Code** — Search "Claude" in the VS Code Extensions tab. Adds the orange Claude icon; sign in and you're working against any opened folder.
- **Warp** — https://www.warp.dev/ — Terminal built for agentic development. Multiple past-conversation view, project explorer, team file sharing, native MCP config.
- **Poppy AI / Claude Canvas** — https://poppy.ai/ — Canvas-style interface with file explorer and multiple Claude Code sessions side by side. Samin wrote this course inside it. <!-- verify URL -->
- **Anthropic Agent Manager** — https://platform.claude.com — Anthropic's cloud agent manager. Agents, sessions, environments, built-in skills (PowerPoints, Excel, PDFs, Docs). Pay-per-API-use.
- **Claude Code (terminal)** — https://claude.com/product/claude-code — Same Claude, different door. `/remote control` syncs terminal sessions back to desktop.

## Links referenced

## Downloads / templates for this day

Enable terminal → desktop sync from inside a Claude Code terminal session:

```
/remote control
```

Then pick "enable remote control from the session." Reload the desktop app, you'll see the same session pinned.

Create your first cloud agent in Anthropic's Agent Manager quick start:

```
Hey, I want you to create an agent that creates YouTube scripts for me, using this skill: [paste your Ralph Wiggum Copywriter skill or whichever one you're using].
```
