# Day 17 — Pull Your Life Into One Place

## Tools used in the video

- **ChatGPT** — [chatgpt.com](https://chatgpt.com) — export your old AI chats from settings → data controls → export data.
- **Composio** — [composio.dev](https://composio.dev) <!-- verify URL --> — free connector hub that plugs Gmail, Slack, Calendar, Drive, Stripe, QuickBooks, Fathom, Fireflies, and a bunch more into Claude with one auth.
- **Claude Desktop** — [claude.ai/download](https://claude.ai/download) — where you'll install the Composio connector and run the "build my second brain" prompt.
- **Obsidian** — [obsidian.md](https://obsidian.md) — where the evolving brain folder lives locally so you can browse and edit it.
- **Fathom** — [fathom.video](https://fathom.video) <!-- verify URL --> — meeting recorder Samin uses to pipe meeting transcripts into the brain every week.
- **Fireflies** — [fireflies.ai](https://fireflies.ai) — alternative meeting recorder with a Composio connector.
- **GitHub** — [github.com](https://github.com) — where you push your brain folder so remote scheduled tasks can reach it.

## Links referenced

- ChatGPT export path: Settings → Data controls → Export data → Confirm export (email arrives with a download link)
- Composio connect apps page (once signed in)
- Claude customize → connectors → add custom connector

## Downloads / templates for this day

Starter prompt for building the second brain:

```
Hey, can you use Composio and in this evolving brain make a bunch of markdown files
such that you collect all the information from my YouTube, maybe get the transcripts
put it there, and essentially build my second brain. Go through all the apps I have
connected in Composio and then make me my brain.
```

Starter prompt for the weekly sync scheduled task:

```
Hey, can you set up a scheduled task so that every week on Saturdays at 6 pm,
I want you to pull down all my meeting transcripts using Composio and Fathom.
Pull down the transcripts, save them in here, and then process them so they live
in my second brain and it all gets indexed. Use all the skills you need to.
```
