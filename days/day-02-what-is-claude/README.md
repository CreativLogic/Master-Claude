# Day 2 — What Is Claude + Your First Setup

> **Pain:** You don't know which Claude model to pick, which plan to pay for, or how to set it up properly.
>
> **Action:** Learn the three Claude models, pick your plan, install the desktop app, and turn on memory + artifacts.
>
> **Time:** ~15–20 minutes · **Video:** `02:18–09:57`

---

## Today's files

- [**🎯 worksheet.html**](worksheet.html) — **interactive worksheet** (open in a browser) with a checklist, reflection prompts, and save/export
- [`lesson.md`](lesson.md) — the written lesson
- [`resources.md`](resources.md) — tools, links, copy-paste prompts
- [`video-section.md`](video-section.md) — transcript slice with timestamps
- [`prompts/`](prompts/) — individual prompt files you can copy
- [`artifact/`](artifact/) — where you save what you build today

## How to start

```bash
# from this folder:
claude
```

That opens Claude Code with the per-day tutor loaded. Ask it anything about today.

## When you're done

From the repo root:
```bash
./scripts/mark-done.sh 2
```

Or check the box in `PROGRESS.md` by hand.
