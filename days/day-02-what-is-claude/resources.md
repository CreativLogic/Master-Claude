# Day 2 — What Is Claude + Your First Setup

## Tools used in the video

- **Claude.ai** — https://claude.ai — the web app. Free to start. Where everyone begins. We're graduating off of this today.
- **Claude Desktop app** — https://claude.ai (bottom-left download icon once logged in) — the full-featured Mac/Windows app. 80% of the course runs here.
- **Claude Pro / Max plans** — https://claude.ai/upgrade — flat monthly subscription, rolling rate limits that reset every few hours or weekly. Pro (~$20/mo) to start. Max ($100 or $200) if you're pacing hard.
- **Claude API usage dashboard** — https://platform.claude.com/usage — pay-as-you-go token usage for developers. Skip this unless you're wiring Claude into software.
- **Whisperflow (optional)** — https://wisprflow.ai — lives in the background on your computer. Hold a key, talk, it drops what you said into whatever app you're in. Faster than typing.

## Links referenced

- `claude.ai/upgrade` — plan comparison and checkout.
- `platform.claude.com/usage` — API usage dashboard (informational only — you don't need this for the course).
- In-app: `Settings → Capabilities` — where memory and artifacts toggles live.
