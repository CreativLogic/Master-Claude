# Day 9 — Share What You Built With Others

## Tools used in the video

- **Claude Cowork + Customize panel** — where you browse, install, and create plugins. Open Claude → Customize → Browse plugins. <!-- verify URL -->
- **Anthropic's built-in plugin marketplace** — marketing, sales, legal starter plugins shipped by Anthropic. Accessed inside **Customize → Browse plugins**. <!-- Samin, verify -->
- **Finder (macOS)** — every project you work in (e.g., `PNL for boss`) lives as a real folder on your Mac. Search the project name to find the plugins and assets Claude saved.

## Downloads / templates for this day

The three-plugin prompt Samin dictated on camera (edit for your own teams):

```
I want you to build me three plugins from the skills we've already created. Here's how to break them up.

Plugin 1 — Core Toolkit (shared base every team gets):
- Branded deck generator (uses our brand guidelines and colors)
- Data puller (Drive, HubSpot, PostHog) — inherit from the weekly report skill
- Required connectors baked in

Plugin 2 — Marketing Ops:
- Inherit everything from the weekly marketing report (full GCPS framework)
- Competitive brief skill
- Content performance skill

Plugin 3 — Sales Ops:
- Pipeline report (pulls weekly sales pipeline, makes a deal deck)
- Deal stage analyzer (flags stuck deals, recommends next actions)
- Follow-up drafter (personalized emails based on HubSpot CRM activity)

For each plugin, create the folder structure with the skill MD files for every skill. Include the brand guidelines and templates. List all required connectors. Make each skill actually work.
```
