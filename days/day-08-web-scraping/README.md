# Day 8 — Let Claude Scrape the Web for You

> **Pain:** Some of the sites your job depends on have no Claude connector — and Claude can't visit them raw.
>
> **Action:** Give Claude a safe browser it can use, so it can scrape hotel prices, product listings, anything — and put it on a schedule.
>
> **Time:** ~15–20 minutes · **Video:** `54:00–1:04:00`

## How to start

```bash
# from this folder:
claude
```

That opens Claude Code with the per-day tutor loaded. Ask it anything about today.

## When you're done

From the repo root:
```bash
./scripts/mark-done.sh 8
```

Or check the box in `PROGRESS.md` by hand.
