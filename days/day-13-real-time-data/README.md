# Day 13 — Give Claude Eyes on Your Data

> **Pain:** Your automations run blind — no real signal, just gut feeling baked into prompts.
>
> **Action:** Wire Claude into a real-time data feed so it can make decisions on actual signals.
>
> **Time:** ~15–20 minutes · **Video:** `1:42:30–1:52:30`

```bash

claude
```

That opens Claude Code with the per-day tutor loaded. Ask it anything about today.

From the repo root:
```bash

```

Or check the box in `PROGRESS.md` by hand.
