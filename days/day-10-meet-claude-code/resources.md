# Day 10 — Meet Claude Code — The Power Version

## Tools used in the video

- **Claude Code (desktop app)** — Claude running on your operating system with direct access to your files. Click the **Code** tab at the top of the Claude desktop app. <!-- verify URL -->
- **GitHub** — public library for code and skills. Free account, sign up with Google. https://github.com
- **Finder (macOS)** — how you browse the folders Claude Code is working inside. Every project is a real folder on your Mac.
- **Opus (model)** — switch to Opus for beefy multi-step builds. Sonnet is for quick, easy commands.
- **Customize panel** — where your skills and connectors live. Same panel you used in Cowork, works in Claude Code too.
