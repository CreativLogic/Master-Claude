# Day 18 — Talk to Claude From Telegram and Discord

## Tools used in the video

- **Bun** — [bun.sh](https://bun.sh) — fast JavaScript runtime the Telegram/Discord plugins need under the hood.
- **Claude Code** — [claude.com/code](https://claude.com/code) <!-- verify URL --> — where you install the Telegram and Discord plugins.
- **Telegram** — [telegram.org](https://telegram.org) — messaging app where you'll pair your first channel.
- **BotFather** — search `@BotFather` inside Telegram — the official Telegram bot that creates new bots and hands you a token.
- **Discord** — [discord.com](https://discord.com) — second channel to pair Claude to.
- **Discord Developer Portal** — [discord.com/developers/applications](https://discord.com/developers/applications) — where you create the bot, grab the token, and set intents.
- **Obsidian** — [obsidian.md](https://obsidian.md) — your evolving brain lives here; you'll see it light up as Claude answers messages.
- **Jump Desktop** — [jumpdesktop.com](https://jumpdesktop.com) — screen sharing software Samin mentions for accessing a Mac Mini from anywhere ($15 one-time).
- **Claude Club (Samin's community)** — [skool.com/claude](https://skool.com/claude) <!-- Samin, verify URL --> — where the setup guide and help lives if you get stuck.

## Links referenced

- Discord developer portal → Applications → New Application → Bot tab → Reset Token
- Discord developer portal → Installation → copy install link → paste in a new tab → authorize
- Discord user settings → Developer mode → turn on (so advanced options show up)
- BotFather `/newbot` flow (pick a name, get a token)
- Claude Code commands used: `claude`, `/plugin install telegram`, `/plugin install discord`, `/discord configure`, `/exit`
